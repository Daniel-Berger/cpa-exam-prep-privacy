- **App Version**: Which version of the App you're using

- **IP Address**: Used temporarily for geolocation (not stored)

- **Screen Views**: Which screens you visit in the App
- **Button Taps**: Which features you use (e.g., "Started Practice Test", "Completed Flashcard Session")
- **Session Duration**: How long you use the App per session
- **Feature Usage**: How often you use specific study modes (flashcards, tests, learn mode)
- **App Performance**: Loading times, responsiveness

### 2.6 Crash & Performance Data
When Firebase Crashlytics is enabled, we automatically collect:
- **Crash Reports**: Stack traces and error logs when the App crashes
- **Performance Metrics**: App launch times, network request durations
- **Device State**: Memory usage, battery level, network type at time of crash

### 7.2 Authentication Security
